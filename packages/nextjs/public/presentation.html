Rootstock Payroll Vault

Smart Contract-Based Employee Salary Management

Built on Rootstock Testnet

Problem & Solution

Problem

  • Traditional finance salary processing can take time and could consume considerable fees in some place

Solution

  • Smart contract-based payroll
  • Vault Funding by Employer
  • Configurable withdrawal day
  • Real-time vault tracking
  • Employee self-service withdrawals

Smart Contract Architecture

┌─────────────────────────────────────────────────────┐
│              VaultRegistry Contract                 │
│  • Maps company → vault address                     │
│  • One vault per company                            │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│             PayrollVault Contract                  │
│  • deposit()        : Add funds to vault           │
│  • addEmployee()   : Register employee + salary    │
│  • removeEmployee(): Remove employee               │
│  • withdraw()       : Employee withdrawal          │
│  • freezeVault()   : Lock all withdrawals          │
│  • setWithdrawalDay() : Configure pay day (1-28)   │
└─────────────────────────────────────────────────────┘

Key Features: RBTC native currency • Monthly salary (with/without first-month salary) • Configurable payment day

Web Application Features

Admin Dashboard

  • Create payroll vault
  • Add/remove employees with salary terms
  • Deposit funds to vault
  • View total monthly obligations
  • Coverage indicator (months of runway)
  • Freeze/unfreeze vault

Employee Portal

  • Enter company vault address
  • View withdrawable amount
  • Withdraw on designated day
  • Track payment history

Tech Stack

  • Next.js 14 + React
  • Wagmi + RainbowKit
  • MongoDB (off-chain indexing) - in progress

How to Use

  1. Connect Wallet
  2. Admin Flow:
    • Navigate to /admin
    • Create new vault
    • Add employees with salary & start date
    • Deposit RBTC to fund the vault
  3. Employee Flow:
    • Navigate to /employee
    • Enter company vault address
    • View withdrawable amount
    • Withdraw on designated day
